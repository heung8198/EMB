Hands up (기지개)가 감지되면 방 창문이 열리고

방문 밖을 나가면 초음파 센서를 지나가면서 거실 led 점등

아침에 준비를 끝내고 현관문을 열고 나가면 열었던 창문, 불들이 자동으로 꺼지는 시스템을 구현했습니다.
